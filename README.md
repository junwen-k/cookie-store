# @cookie-store

Reactive bindings for the [Cookie Store API](https://developer.mozilla.org/en-US/docs/Web/API/Cookie_Store_API). Supported in React, Vue, Svelte, Solid, Preact, and Angular.

## Features

- 🍪 **Reactive reads** — Automatic updates when cookies change
- 🛠️ **Type-safe** — First-class TypeScript support
